This repository Jenkins server using the free AMI from Bitnami. https://docs.bitnami.com/aws/apps/jenkins

JenkinsBase.yaml: Launches VPC, Subnets, Security Groups and IAM entities. Most other stacks depend on this one.

JenkinsServer.yaml: It creates an Jenkins Server
Log on as the "bitnami" user, the password is randomly autogenerated and can be
retrieved from the AWS EC2 web interface:
click on the instance -> Actions -> Instance Settings -> Get System Log.
